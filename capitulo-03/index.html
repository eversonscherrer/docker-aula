Docker vs Maquina Virtual

Apartamento ou casa? Qual a melhor para cada situacao

🏠 VM = A casa

  • SO proprio (kernel completo)
  • Drivers, libs, bibliotecas
  • Tudo isolado do host
  • Gasta GB de RAM e disco
  • Demora minutos pra ligar

📦 Container = O apartamento

  • Compartilha o kernel do host
  • So leva libs proprias do app
  • Isolado, mas leve
  • Gasta MB de RAM, nao GB
  • Liga em segundos
