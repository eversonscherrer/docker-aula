Historia do Docker

De projeto interno a revolucao mundial em 10 anos

2013

Nasce o Docker

Solomon Hykes apresenta o Docker no PyCon, Santa Clara, em apenas 5 minutos. Era uma ferramenta interna da dotCloud — decidiram abrir o codigo.

2014

As gigantes adotam

Google, Microsoft e IBM comecam a apoiar e integrar Docker em seus servicos de cloud.

2015

Kubernetes e Docker Compose

Surgem ferramentas para orquestrar varios containers ao mesmo tempo. O ecossistema explode.

2017

Docker no Windows

Docker chega nativamente ao Windows, expandindo seu alcance para alem do Linux.

2020

Pandemia acelera a cloud

A migracao massiva para trabalho remoto e cloud consolida Docker como padrao da industria.

2025+

Impossivel ignorar

E praticamente impossivel trabalhar com TI sem encostar em Docker. Virou habilidade essencial.
